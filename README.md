## Installation

```bash
$ npm install
```

## Running the DockerCompose

```bash
$ Docker-compose up -d
```

## Running the Prisma migrate for create database

```bash
$ npm run  migrate:all

```

## Running the dataseed for Initail Data

```bash
$ npm run dataseed


```

## Running the build app

```bash
# build app all
$ npm run build:all

```

## Running the app

```bash
# run  app all
$ npx pm2 start ecosystem.config.js
```
